# WEB ALPHA Book Store

## Project Description
WEB ALPHA Book Store is an interactive online bookstore web application. Users can browse featured books, view book details, search for books, like books, add books to the cart, place orders, and submit reviews. The project also includes an admin dashboard for managing books, users, orders, reviews, prices, discounts, ratings, and book visibility.

## How to Run the Project Locally
1. Download or clone the repository.
2. Open the project folder.
3. Make sure all files and the `image` folder are in the same project directory.
4. Open `index.html` in a web browser.
5. To test user features, create a new account using the Sign Up form or use an existing user account.
6. To test the admin dashboard, log in with the admin account below.
7. After logging in as admin, open `admin.html` or click the Admin Page link.
